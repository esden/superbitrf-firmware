superbitrf-firmware
===================

Firmware for the superbit open source radio system.


Compiler:
========

e.g. default paparazzi compiler:

PREFIX=/opt/paparazzi/arm-multilib/bin/arm-none-eabi

e.g. summon arm toolchain:

PREFIX=~/sat/bin/arm-none-eabi
